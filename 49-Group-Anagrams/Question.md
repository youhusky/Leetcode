# 49. Group Anagrams

[Original Page](https://leetcode.com/problems/anagrams/)

Given an array of strings, group anagrams together.

For example, given: `["eat", "tea", "tan", "ate", "nat", "bat"]`,  
Return:

<pre>[
  ["ate", "eat","tea"],
  ["nat","tan"],
  ["bat"]
]</pre>

**Note:** All inputs will be in lower-case.
